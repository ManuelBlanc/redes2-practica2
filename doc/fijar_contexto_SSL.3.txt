
FIJAR_CONTEXTO_SSL(3)
=====================
:doctype: manpage
:format: manpage

NOMBRE
------
*fijar_contexto_SSL* - Crea un contexto SSL

SYNOPSIS
--------
*#include <G-2301-05-P3-ssl_functions.h>*

_Redes2_SSL_CTX*_ *fijar_contexto_SSL*(_Redes2_SSL_CTX_config cnf_);

DESCRIPTION
-----------
Esta funcion se encarga de inicializar correctamente el contexto que es utilizado para la creacion de canales seguros mediante SSL. Recibe un objeto _Redes2_SSL_CTX_config_ con informacion sobre las rutas a los certificados y claves con los que vaya a trabajar la aplicacion.

RETURN VALUE
------------
*NULL*::    La creacion del contexto SSL ha fallado
*Puntero*:: Puntero a un contexto inicializado

SEE ALSO
--------
inicializar_nivel_SSL(3), aceptar_canal_seguro(3), cerrar_canal_seguro(3)
