
SSOCK_GET_FD(3)
===============
:doctype: manpage
:format: manpage

NOMBRE
------
*ssock_get_fd* - Devuelve el descriptor de fichero del SSock

SYNOPSIS
--------
*#include <G-2301-05-P3-ssock.h>*

_int_ *ssock_get_fd*(_SSock* ss_);

DESCRIPTION
-----------
Devuelve el numero asociado al SSock _ss_ proporcionado. Esta funcion debe usarse con precaucion, puesto que puede haber un objeto SSL por encima del descriptor que haga asunciones sobre el socket.

RETURN VALUE
------------
*>0*::  El descriptor de fichero
*ERR*:: Ha ocurrido un error (argumento erroneo).
